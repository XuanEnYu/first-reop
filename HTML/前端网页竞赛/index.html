什么是前端？

前端，通常指的是网站的前台页面，也就是用户可以直接与之交互的部分。前端开发者使用HTML、CSS、JavaScript、jQuery等技术来构建网站的前端部分，包括网站的结构、布局、内容、交互等。前端开发者需要掌握HTML、CSS、JavaScript、jQuery等技术，并熟练使用前端框架、库、工具，如Bootstrap、React、Vue等。

网站前端

前端技术一般分为前端设计和前端开发，前端设计一般可以理解为网站的视觉设计， 前端开发则是网站的前台代码实现，包括基本的HTML和CSS以及JavaScript/ajax， 最新的高级版本HTML5、CSS3，以及SVG等。

核心技术

这三个是前端开发中最基本也是最必须的三个技能。前端的开发中，在页面的布局时， HTML将元素进行定义，CSS对展示的元素进行定位，再通过JavaScript实现相应的效果和交互。 虽然表面看起来很简单，但这里面需要掌握的东西绝对不会少。在进行开发前，需要对这些概念弄清楚、 弄明白，这样在开发的过程中才会得心应手。

HTML

HTML (Hypertext Markup Language) 超文本标记语言，它是一种用于创建网页的标记语言。 HTML 文档结构由标签和元素组成，标签用于定义元素的属性，元素则是网页的基本构成单位。 HTML 文档可以嵌入各种外部文件，如图片、视频、音频等。HTML5 则是 HTML 的最新版本， 它新增了许多特性，如语义化标签、音视频、地理位置、拖放、Web 存储、Web 应用等。

CSS

CSS (Cascading Style Sheets) 层叠样式表，它是一种用来表现 HTML 或 XML 文档样式的计算机语言。 CSS 用来控制网页的布局、配色、字体、边框、动画、多媒体等。CSS3 则是 CSS 的最新版本， 它新增了许多特性，如圆角、边框阴影、渐变、多列布局、网格布局、文本效果、过渡效果、变换效果等。

JavaScript

avaScript 是一种动态的脚本语言，它可以实现网页的各种动态功能。它可以控制网页的各种元素， 包括HTML、CSS、DOM、BOM等，还可以实现动画、音频、视频、地图、图表、轮播图、表单验证等功能。
